# DevToys TUI

A combination of commonly use tools, in the terminal.

## How to use

1. Ensure Golang is installed: [https://go.dev/doc/install](https://go.dev/doc/install)

2. Install the tool:

```shell
go install github.com/ducthuy-ng/devtoys-tui@latest
```

3. Use the tool:

```shell
devtoys-tui
```
